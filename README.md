# Krishi Bangla (কৃষি বাংলা)

Krishi Bangla is an interactive, bilingual (English/Bengali) web platform dedicated to showcasing the rich agricultural landscape of Bangladesh. Nestled in the world's largest river delta, Bangladesh's agriculture feeds over 170 million people. This project brings that story to life through interactive maps, data charts, and AI-powered tools.

## 🌾 Features

- **Bilingual Interface:** Seamless toggling between English and Bengali across all components.
- **Detailed Agricultural Overview:** Insights into Rice Production, Fisheries, Jute Export (The Golden Fiber), and Climate Risks.
- **Interactive Crop Map:** A visualization of where and what crops are grown using `d3-geo`.
- **Weather Widget & Climate Data:** Up-to-date weather and climate data integration.
- **AI Chat Assistant:** Powered by Google Generative AI and OpenAI for answering agriculture-related queries.
- **Modern UI:** Built with Tailwind CSS and Next.js for a responsive, fast, and sleek experience.

## 🛠 Tech Stack

- **Framework:** [Next.js](https://nextjs.org) (App Router)
- **Styling:** Tailwind CSS (v4)
- **Data Visualization & Maps:** D3 Geo
- **AI Integrations:** Google Generative AI (@google/generative-ai), OpenAI
- **Typography:** Geist Fonts

## 🚀 Getting Started

First, ensure you have your environment variables set up. You will need API keys for Google Generative AI and/or OpenAI if you are using the AI Chat features.

To run the development server:

```bash
npm install
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
